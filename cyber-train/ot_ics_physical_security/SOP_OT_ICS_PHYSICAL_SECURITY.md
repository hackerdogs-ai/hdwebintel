# Standard Operating Procedure (SOP) — OT/ICS/Physical Security Pillar

## 1. Overview & Purpose

OT/ICS/Physical Security ensures **secure, protected, and monitored** operational technology, industrial control systems, and physical infrastructure across the enterprise. This SOP protects against OT/ICS attacks, physical security breaches, and ensures adherence to OT/ICS security best practices, physical access controls, and regulatory requirements.

**Key Threats Addressed:**
- OT/ICS attacks and industrial espionage
- Physical security breaches and unauthorized access
- OT/ICS vulnerabilities and exploitation attempts
- Physical infrastructure failures and sabotage
- OT/ICS network segmentation violations and policy breaches
- Physical access control failures and credential abuse
- OT/ICS monitoring gaps and visibility issues
- Physical security compliance violations and audit failures
- OT/ICS supply chain attacks and component compromise
- Physical security incidents and emergency response failures

---

## 2. Scope

* All **operational technology (OT)** systems across the enterprise
* All **industrial control systems (ICS)** and SCADA systems
* All **physical security** infrastructure and access controls
* All **OT/ICS compliance** and regulatory frameworks
* All **OT/ICS/Physical security testing and validation**

---

## 3. Roles & Responsibilities

### **Strategic Roles**

* **Chief Information Security Officer (CISO)** → defines OT/ICS/Physical security strategy, risk appetite, and governance
* **Principal OT/ICS Security Architect** → designs comprehensive OT/ICS security architecture and controls
* **Head of OT/ICS/Physical Security / Industrial Security Manager** → owns operational execution and compliance
* **Chief Technology Officer (CTO)** → ensures OT/ICS security integration in development processes
* **Head of Risk & Compliance** → ensures adherence to OT/ICS/Physical security standards and regulations

### **Execution Roles**

* **OT/ICS Security Engineer** → implements OT/ICS security controls, monitoring, and protection
* **Physical Security Specialist** → manages physical security infrastructure and access controls
* **OT/ICS Network Specialist** → manages OT/ICS network segmentation and isolation
* **OT/ICS Monitoring Specialist** → manages OT/ICS monitoring and threat detection
* **OT/ICS Security Tester** → conducts OT/ICS security testing and vulnerability assessments
* **OT/ICS Compliance Analyst** → ensures OT/ICS/Physical security compliance and regulatory adherence
* **Emergency Response Specialist** → manages emergency response and crisis management
* **Incident Responder (OT/ICS/Physical Security)** → handles OT/ICS/Physical security breaches and incidents
