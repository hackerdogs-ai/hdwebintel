Some of the best modern open-source products for incident response and investigation are TheHive for case management, Velociraptor for digital forensics, OSSEC or Wazuh for host-based intrusion detection, Snort or Suricata for network intrusion detection, and Security Onion for a comprehensive security monitoring platform. Other notable tools include OSQuery for system querying, Wireshark for network analysis, and OpenVAS for vulnerability scanning. [1, 2, 3, 4]  
Incident management and collaboration 

• TheHive: A scalable platform for collaborative investigation and case management. [1, 5]  
• IRIS: A web-based collaborative platform for incident response analysts to share investigations. [6]  
• Security Onion: A comprehensive Linux distribution for threat detection, security monitoring, and log management. [3, 4]  

Digital forensics and endpoint analysis [1, 6]  

• Velociraptor: A tool for digital forensics and incident response, allowing for deep endpoint inspection. [1, 7, 8]  
• OSQuery: Turns a system into a relational database, allowing you to use SQL to query system state and behavior. [1, 4]  
• Limacharlie: A cross-platform endpoint security platform that can push modules into memory to extend functionality. [6]  

Network security and intrusion detection 

• Snort / Suricata: Widely used, high-performance network intrusion detection/prevention systems (IDS/IPS). 
• OSSEC / Wazuh: Host-based intrusion detection systems (HIDS) for detecting malicious activity on endpoints. 
• Wireshark: A network protocol analyzer used for deep packet inspection and analysis. 
• Nmap: A network mapping and security auditing tool. [1, 3, 4, 9, 10, 11]  

Vulnerability and threat assessment 

• OpenVAS: A widely used open-source vulnerability scanner. 
• Nikto: A web server scanner that checks for outdated software and common misconfigurations. [1, 3, 4]  

Other notable tools 

• MISP: A platform for sharing threat intelligence. 
• Maltego: A tool for open-source intelligence (OSINT) investigations. [12, 13, 14]
