## How to Use These Diagrams

Each diagram shows:
- **Metrics** (top section): Specific KPIs and measurements for the dashboard
- **Data Sources** (middle section): Tools and platforms that provide the data
- **Collection Methods** (bottom section): Techniques used to gather and process the data

Arrows connect metrics to their data sources, and data sources to their collection methods, providing a complete view of the data flow.
